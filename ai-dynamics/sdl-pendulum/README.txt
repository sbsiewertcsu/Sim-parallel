Double Pendulum - Real-Time SDL2 Visualization
==============================================

This is a real-time simulation of a double pendulum using C++ and SDL2.

Requirements:
-------------
- C++ compiler (g++)
- SDL2 development libraries

Install SDL2 on Ubuntu/Debian:
------------------------------
sudo apt update
sudo apt install libsdl2-dev

Build and Run:
--------------
make
./double_pendulum_sdl

Files:
------
- double_pendulum_sdl.cpp : Main simulation code
- Makefile                 : Build instructions
- README.txt               : This file
